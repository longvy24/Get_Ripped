# Get Ripped — 120-Day Plan

A self-contained, weight-reactive 120-day fat-loss program with full workout tracking, meal planning, supplement protocol, and shopping lists. Runs entirely in your browser. No accounts, no servers, no tracking.

🌐 **Live site:** https://longvy24.github.io/Get_Ripped/

## What's in here

A single-page web app that includes:

- **Training** — 4-phase, 5-day-a-week program (Ramp-Up → Build → Push → Peak/Cut). Two-a-day schedule built for a 9-5 worker (AM cardio + PM lift). Per-exercise form cues and YouTube search links. Check-marks and weight×reps logging for every set.
- **Macros** — Daily calorie/protein/carb/fat targets that auto-scale to your current bodyweight.
- **Meals** — 7-day meal rotation built from chicken, beef, pork, eggs, dairy, rice, pasta, bread, potatoes, and fruit in shakes. Every meal's macros scale to your weight. Fallback schedules for 4-meal, 3-meal, and 2-meal days.
- **Supplements** — Daily protocol, brand recommendations, monthly cost comparison.
- **Progress** — Weight log with trend chart, body measurements tracking, automatic adjustment alerts.
- **Shopping** — Three budget tiers (Rock Bottom / Budget / Comfortable) × two store options (warehouse club / regular supermarket) with live cost totals.
- **Settings** — Export/import all your data as JSON for backup. Theme toggle, reset options.

## How to use

Just open `index.html` in any modern browser. Everything is local — your data is stored in browser localStorage and never leaves your device.

### Install as an app
- **Chrome / Edge** — Click the install icon in the address bar, or menu → Install Get Ripped.
- **Safari (Mac)** — File → Add to Dock.
- **iPhone / iPad** — Share → Add to Home Screen.

### Backup your data
Settings tab → Export My Data. Save the JSON file in Dropbox/iCloud. You can re-import it later or move it to another browser/device.

## Tech

- One HTML file. Zero dependencies. No build step.
- ~120 KB total. Loads instantly. Works offline once cached.
- localStorage for persistence. SVG charts. No third-party trackers.
